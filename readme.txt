== Description ==

This plugin add a flying twitter bird in your blog. If you move the mouse over the bird, 
a "tweet this" -and displays a "follow me" button. The bird found itself as the "follow me" link on the Twitter account.
In addition you can also  chage color of bird  according to your mood ,at backend options are present to change color of bird an set your twitter page name.




== Installation ==

1. Upload the `flyingtwitter` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu
3. Navigate to 'Flying Twitter' menu from admin panel
